## 一、前言

農產品交易資料分析系統(網頁版)（以下簡稱「本系統」）由 VNAos 團隊、鹿中創客 開發與維護。
我們重視您的隱私權，並致力於保護您的個人資料，故建立本隱私權政策。
本隱私權政策將說明我們如何收集、使用、儲存和保護您的資訊。

**使用本系統即表示您同意本隱私權政策的內容。** 如果您不同意本政策的任何部分，請立即停止使用本系統。

## 二、資料收集

### 2.1 我們收集的資料類型

本系統為**純前端應用程式**，原則上**不會主動收集**您的個人資料。我們可能收集或處理的資料包括：

#### 自動收集的技術資料
- **瀏覽器資訊**：您的瀏覽器類型、版本、作業系統
- **裝置資訊**：裝置類型、螢幕解析度
- **使用記錄**：頁面瀏覽記錄、存取時間（透過瀏覽器自動記錄）
- **IP 位址**：
  - 用於連線至 API 伺服器時可能被記錄（非本系統紀錄）
  - 用於連線至 GitHub Pages 時可能被 GitHub 記錄（非本系統紀錄）

#### 使用者輸入資料（僅在瀏覽器端處理）
- **搜尋關鍵字**：您在搜尋欄位輸入的作物名稱（僅在瀏覽器端處理，不傳送至伺服器）
- **選擇偏好**：您選擇的市場和作物（僅儲存在瀏覽器端，不會傳送至伺服器）

### 2.2 我們不收集的資料

本系統保證**不會收集**以下資料：
- 姓名、電子郵件、電話號碼等個人識別資訊
- 位置資訊（GPS 座標）
- 帳號密碼
- 付款資訊
- 其他個人敏感資料

另外，本系統保證**不會**使用以下方式詢問您的個人資料:
- 致電
- Email
- 任何通訊軟體
- 其他任何方式

本系統保證不蒐集您的個人資料。

我們使用收集到的資料僅用於以下目的：

1. **提供服務功能**：顯示農產品交易資料和分析圖表

3. **技術維護**：確保系統正常運作，排除技術問題
4. **統計分析**：進行匿名化的使用統計（如總訪問次數）

## 五、資料儲存與保留

### 5.1 資料儲存位置

- **前端資料**：所有使用者輸入和選擇僅儲存在您的瀏覽器中
- **API 資料**：農產品交易資料由 API 伺服器提供，我們不儲存這些資料
- **網站檔案**：本系統的靜態檔案儲存在 GitHub 伺服器上（公開儲存庫）
- **技術記錄**：
  - API 伺服器可能記錄連線記錄，但我們無法存取這些記錄
  - GitHub Pages 可能記錄網站存取記錄，但我們無法存取這些記錄

### 5.2 資料保留期限

- **瀏覽器端資料**：由您的瀏覽器設定決定，可隨時清除
- **API 伺服器記錄**：不特地紀錄
- **GitHub Pages 記錄**：由 GitHub 的隱私權政策決定，我們無法控制

### 5.3 未來可能的快取機制

為提升效能，未來版本可能會使用瀏覽器的 `localStorage` 或 `sessionStorage` 快取資料。這些資料：
- 僅儲存在您的裝置上
- 不會傳送至任何伺服器
- 可隨時透過瀏覽器設定清除

## 六、資料安全

### 6.1 安全措施

我們採取以下措施保護您的資料：

1. **HTTPS 傳輸**：所有資料傳輸均使用加密連線
2. **前端處理**：敏感操作盡可能在瀏覽器端完成
3. **最小化收集**：僅收集提供服務所必需的資料
4. **定期更新**：保持系統和依賴套件的最新版本

### 6.2 資料安全限制

請注意：
- 本系統為靜態網站，不設有任何後端伺服器
- 本系統透過 GitHub Pages 託管，網站檔案儲存在 GitHub 伺服器上
- 我們無法完全控制第三方服務（如 GitHub Pages、API、CDN）的資料處理方式
- 建議您使用安全的網路環境存取本系統
